Here are some sample labs to try

### Even Odd Can you turn on only odd pixels?

Hint: Try using the modulo function (%).

### Fade In and Out Can you make a pixel slowly fade in and out?

Hint: Use a for loop to change the brightness.

### Random Can you use the random() function to randomly turn various pixels on? Can you also use random() to select a random color?

### Movement Can you make a pixel appear to move up and down?

Hint: use a for loop to turn a pixel on and off.

### Bubble Can you make a pixel appear to bubble up?

### Twinkle Can you make a random pixel appear to twinkle on and off?

### Heartbeat Can you make a pixel appear to pulse like a beating heart?
