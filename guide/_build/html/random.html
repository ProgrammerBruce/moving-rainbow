# Random

Lets face it. Computers are really good at repeating the same pattern over and over. But after a while this gets pretty boring. What might be fun is to add a little bit of random behavior into our patterns. Luckily the Arduino software has just the function for us! It is called, you guessed it [random](http://arduino.cc/en/reference/random).
